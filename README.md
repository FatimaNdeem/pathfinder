# pathfinder
An interactive Python application that brings pathfinding algorithms to life! Watch in real-time as A* and Greedy Best-First Search navigate through dynamically changing mazes, complete with a live metrics dashboard showing nodes visited, path costs, and execution times. Built with Pygame, this educational tool lets you toggle between Manhattan and Euclidean heuristics, create and destroy walls with a click, and observe how different algorithms find their way from start to goal. Perfect for students, educators, and anyone curious about how AI pathfinding works under the hood.
